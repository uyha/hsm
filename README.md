# Hierarchical state machine (HSM)

A library for declaring hierarchical state machines.

## Features

- [ ] Initial states
- [ ] States
- [ ] Transitions
- [ ] On entry
- [ ] On exit
- [ ] Actions
- [ ] Event generating actions
- [ ] Self-Transitions
- [ ] Compound states
- [ ] Guards
- [ ] Terminate state
- [ ] History state
- [ ] Concurrent regions
